Category Ruleset Breakdown

This list provides a breakdown of what glitches are allowed in what category ruleset.

Restriction 00

This restriction class allows everything. The glitches and techniques listed here correspond to those that are disallowed by the global rules.

  • RAM preparation
  • Arbitrary Code Execution (ACE)
  • Conflicting directionals (udlr)

Restriction 10

This generally unrestricted classs allows for all glitches and techniques that are not banned by global rules.

  • Arbitrary Ancilla Misslotting (slots 10+)

Restriction 20

This restriction class roughly corresponds to the old "Unrestricted" ruleset. By convention, plaid world is categorically considered EG.

  • Layer Disparities (Exploration Glitch/EG)
  • Plaid World (RAM EG)
  • Ancilla Misslotting (slots 0–9)
    • Hookpush
    • Hookclip

Restriction 30

This restriction class roughly corresponds to the old "Door Glitches" or "No EG" ruleset.

  • Underworld Transition Glitches ("door glitches")
    • Somaria juking
    • Bomb juking
    • Underworld YBA
    • Deathhole
    • Pit wrong warping
    • Quadrant glitch
  • Sprite misslotting (DSSO)
  • Data spoofing

Restriction 40

This restriction class roughly corresponds to the "Restricted Major Glitches" ruleset, previously known as "No EG/DG/WW"

  • Overworld transition glitches
    • Overworld YBA
    • Overworld screen wrap
    • Mirror wrap
  • Wall clips and teleports
  • Door state extension
  • Rail ledge clipping

Restriction 50

This restriction class roughly corresponds to the "No Major Glitches" ruleset.

  • Spinspeed and itemdashing
  • Fake flippers
  • Water walk
  • Fake powder
  • Ancilla overload
  • Mirror block erase
  • Block clipping
  • Silverless Ganon
  • Superbomb/Purple chest dashing
  • Bomb dupe
  • Medallion cancel
  • Mimic clip
  • Spooky action
  • Interior rail clipping
  • Hovering
  • Fire rod doubling
  • Heart dupe
  • Somaria magic refill
