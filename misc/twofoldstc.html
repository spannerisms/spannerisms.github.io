This table supplements the WEST SOMARIA Explication with documentation of all the twofold-corruption candidate rooms jsd found. I've simply included everything interesting that might potentially be corrupted. I have not verified anything, but I have removed the following rooms:

  • Room 029 - Mothula's arena: has no doors
  • Room 089 - Eastern Palace fairies: has no doors
  • Room 0A6 - Moldorm 2 fall: has no doors
  • Room 121 - Blacksmith's: has no suitable doors

Being listed here doesn't necessarily mean it's possible to do both flavors of somaria transition corruption required. Nor does it mean it's possible to set up the WEST SOMARIA variables to make use of any particular entry.

Each write is 16-bit, so every base-offset pair can hit 2 addresses. If you notice some values that don't seem to add up, that's because TMAPX gets bit masked in some of the routines.
